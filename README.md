# Snake-React

This is a very simple snake game made in react. No canvas, every pixel is a `div` element.

## How to run

Install dependencies

```shell
npm install
```

Run in dev mode

```shell
npm run dev
```

## How to play

| Command | Keyboard                       | Virtual Phone |
| ------- | ------------------------------ | ------------- |
| Start   | `Space`                        | `*`           |
| Stop    | `Enter`                        | `#`           |
| Up      | `ArrowUp`/`NumpadUp`/`W`       | `2`           |
| Down    | `ArrowDown`/`NumpadDown`/`S`   | `8`           |
| Left    | `ArrowLeft`/`NumpadLeft`/`A`   | `4`           |
| Right   | `ArrowRight`/`NumpadRight`/`D` | `6`           |
